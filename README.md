🚲 Project Highlights
🤖 AI-powered bike demand prediction using Machine Learning algorithms.
📊 Big Data Analytics to analyze bike-sharing usage patterns.
🛠️ Smart repair optimization by predicting maintenance requirements.
📈 Compared multiple ML models including Random Forest, Decision Tree, KNN, SVM, and Neural Networks.
🌦️ Uses weather, temperature, humidity, date, and historical usage as prediction features.
⚡ Improves repair shop capacity planning with intelligent demand forecasting.
📉 Reduces repair lead time and increases overall system efficiency.
🔄 End-to-end ML pipeline including data preprocessing, training, evaluation, and prediction.
🐍 Built with Python, Scikit-learn, Pandas, and Django.
🌍 Uses real-world Washington D.C. Bike Sharing dataset for model training.
